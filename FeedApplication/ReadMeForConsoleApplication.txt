 Pre-requistes:
 Install .net core 3.1, visual studio
 
 1. Installation steps
   a) Build the project and copy the feeds folder to 'bin\Debug\netcoreapp3.1' folder
   b) There will be a exe file in 'bin\Debug\netcoreapp3.1' folder with name 'FeedApplication.exe'
   c) To run above file, open cmd in same folder and run command 'FeedApplication.exe $ import softwareadvice feed-products/softwareadvice.json'
 2. Where to find your code : FeedApplication\FeedApplication\Program.cs
 3. Was it your first time writing a unit test, using a particular framework, etc? - For console application, have written test cases first time
